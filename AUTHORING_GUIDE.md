# Authoring Guide: Callouts and Rich Content

Use these styles in the editor to highlight important information. They render consistently on the website.

## Callout Blocks

- Callout: general emphasis
- Info: neutral note
- Warning: cautionary content
- Tip: best practice or quick win

### How to apply in the editor (TinyMCE)

1. Select the paragraph(s) you want to convert.
2. Open the Styles dropdown in the toolbar.
3. Choose one of:
   - Callout
   - Info
   - Warning
   - Tip
4. (Optional) Add a bolded first line as the title: select the text and apply the "Callout Title" style or just make it bold.

Example structure inside a callout:

<div class="callout">
  <strong class="callout-title">What this means</strong>
  <p>Short supporting explanation with a link if useful.</p>
</div>

## Inline code and Code blocks

- Use the Code button for inline code.
- Use the Code Sample button for multi-line code blocks.

## Images

- Prefer WebP with descriptive alt text.
- Keep width under ~1200px; the site handles responsive sizing.

## Headings

- Use a logical hierarchy: H2 for major sections, H3 for sub-sections.

## Links

- Use descriptive anchor text. Avoid "click here".
